Learning Javascript and JQuery

  • JavaScript

    • An object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • Operators

    • a symbol that is used to perform an operation, such as addition or substraction.

  • Variable naming conventions

    • Must begin with a letter or an underscore character, not a number. Variable names in javascript are also case-sensitive

  • Functions

    • A block of code designed to perform a task. Written with the function keywork, follwed by a name, followed by a parenthesis.

  • Methods

    • A method is a piece of code that is called by name that is associated with an object.

  • Parameters

    • Variables listed as a part of the function listed in the function definition.Example: function functionName(parameter1, parameter2, parameter3)

  • Return

    • Cease execution in a function and return a value to the caller. Require an explicit return statement for returning the result of an expression (the value) from a function. When a return statement is not present, the interpreter automatically returns undefined.
